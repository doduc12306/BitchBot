BitchBot

Fun Discord bot with moderation and utility tools like reminders, starboard, activity tracking and more

Features

Use any emoji from any server I am in
I let you use any emoji, animated or not, that I can see, no need to have nitro or be in a specific server Even have me delete your command message. I will tell even you send it so you don't end up wondering which ghost bruhed you
Have fun, lots of fun
Get top posts from any subreddit you like, roast people, get jokes, facts and much more Even fetch information from the one and only Urban Dictionary
Track activity of your server members
I allow you to keep track of how much people in your server are active. Active people will get more activity points. Give active people and excuse to flex with their posision activity leaderboard
Play music right in your server
Listen from whatever source you like including Youtube, SoundCloud, Vimeo, Twitch and more while having access to all the features you enjoy
Set whatever prefix you want to
Don't like my default prefix of >? No need to worry. You can completely replace it with whatever you want
Showcase bruh moments in your server
With my starboard, you can archive the bruh moments in a specific channel in your server. With the custom star count, you can let the people star a message and have it archived. Let the members pick what needs to be showcased.
Moderate your server with great moderation tools
Need to yeet someone? Ban someone? Stop someone from being able to talk? I got you covered there You can even add roles as Mod roles which lets people with those roles do even more stuff like warn members
